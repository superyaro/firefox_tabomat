URL-to-Window Router

Configure how new URLs open in existing project or domain windows.

Routing

Routing Rules

Rules are checked top to bottom before domain routing. Use them when one domain contains separate workspaces that should stay in different windows.

Use ! to capture one path segment for the window group and * as the final segment for the rest of the URL.

https://github.com/gorhill/!/* captures repository names such as uBlock.

Inside one segment, !-* captures the part before the dash, so https://youtrack.jetbrains.com/issue/!-* captures IDEA from IDEA-380201.

No custom rules configured.
